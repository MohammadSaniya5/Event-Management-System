#  Event Management System

A web-based Event Management System built using PHP and MySQL that allows users to register for events, make payments, and track their registration status. Admin can manage events and monitor registrations.

---

##  Features

###  User Side
- User Registration & Login
- Browse Available Events
- Register for Events
- Upload Payment Screenshot / QR Payment
- View Registered Events
- Track Payment Status

###  Admin Side
- Admin Login 
- View Total Registrations per Event
- View Complete User Details
- Verify Payment Status

---

##  Tech Stack

- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Hosting: InfinityFree
- Version Control: Git & GitHub

---

##  Database Setup

1. Create MySQL database.
2. Import the provided `registration.sql` file using phpMyAdmin.
3. Update database credentials in `db.php`.
